2 Getting emacs to give you options

ALL THIS STUFF GOES IN YOU .emacs FILE 

;;;;SLIME MODE SETTING INFERIROR LISP
(setq inferior-lisp-program "~/scripts/ccl")
(require 'slime-autoloads)
(setq slime-net-coding-system 'utf-8-unix)
(slime-setup '(slime-fancy))

(setq slime-lisp-implementations
      '((ccl ( "~/scripts/ccl"))
        (sbcl ("/opt/sbcl/bin/sbcl") :coding-system utf-8-unix)))

3 Getting the options working in emacs

3.1 M- - M-x slime (thats Alt minus sign then Alt-x slime)

this command will enter the function above in your .emacs file. from there you will be asked which lisp you want to use
